# AI Resume Analyzer

A web application that analyzes resumes and provides skill insights.

## Features

- Resume Upload (PDF/DOCX)
- Resume Text Extraction
- Skill Detection
- Resume Score Calculation
- Job Role Prediction
- Missing Skill Recommendation

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Project Structure

AI-Resume-Analyzer
│
├── app.py
├── requirements.txt
├── resume_parser
│   ├── parser.py
│   └── skills.py
├── templates
├── static

## How to Run

pip install -r requirements.txt  
python app.py
